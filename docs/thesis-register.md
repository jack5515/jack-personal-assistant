# Thesis Register

这份文档现在就是创业 `thesis board`。用途不是存趋势感想，而是持续收口那些会影响 Jack 选楔子、选用户、选收费方式、选资源分配的核心命题。

目标不是重复 `MEMORY.md`，而是把那些会反复影响巡检、产品判断、创业判断的核心命题单独列出来，方便：
- 被新 signal 更新
- 被反例挑战
- 反向指导下一轮 watchlist
- 直接拉出本周验证动作和不做清单

## Use Rule

只有满足下面至少一条的判断，才适合进入 thesis register：
- 已被多个 signal 反复印证
- 明显影响产品路线、创业方向或资源分配
- 未来 1-4 周内仍值得持续验证

每条 thesis 默认写 `8` 个字段：
- `Thesis`
- `为什么当前成立`
- `支持证据`
- `反证 / 削弱条件`
- `当前资源含义`
- `这周该做什么验证`
- `目前不该做什么`
- `当前状态`

## Thesis List

### 1. 先做强工作流闭环，不先做通用 agent 包装层
- 为什么当前成立：Jack 当前更需要一个能稳定解决高频任务的楔子，而不是一个什么都能聊、但没有明确交付边界的通用壳。对一人公司来说，闭环优先比叙事优先更重要。
- 支持证据：长期背景已经多次收敛到 `context-first productivity agent`、`先强闭环再泛化`；当前巡检目标也越来越聚焦会议后跟进、截图即办、跨端接力、待办推进、知识沉淀这类断裂点。
- 反证 / 削弱条件：如果真实市场证据持续表明，用户更愿意为通用 shell 入口买单，而不是为具体工作流闭环付费和复用。
- 当前资源含义：巡检默认优先找高频断裂点、恢复机制、状态跟踪、默认交付，而不是通用聊天入口或人格包装。
- 这周该做什么验证：继续补 `会议后跟进 / 截图即办 / 跨端接力 / 待办推进 / 知识沉淀` 的强痛点证据、迁移理由和可收费场景。
- 目前不该做什么：不要把纯“大厂新入口”“更强模型”“更会聊天”直接当作强利多。
- 当前状态：高优先级，持续验证中。

### 2. 真正能卖钱的层是服务系统，不是 agent 身份
- 为什么当前成立：模型和 runtime 会越来越接近，真正留住用户的更像 `默认配置 + 交付质量 + 恢复能力 + 状态可视化 + 记忆策略` 的整套服务体验。
- 支持证据：长期判断已经明确是“卖服务，不卖角色”；Claw 赛道里，用户更在意稳不稳、值不值、出了问题有没有人管，而不是 agent 人设是否花哨。
- 反证 / 削弱条件：如果后续真实付费和留存证据表明，用户持续为 agent identity、人格包装或单纯模型能力溢价付费，而不在乎服务层差异。
- 当前资源含义：巡检要优先盯定价页、套餐拆法、默认配置、失败恢复、权限/状态展示、升级口径。
- 这周该做什么验证：继续找谁在卖“少漏事、少折腾、自动推进”的服务结果，谁还在卖“更强模型能力”。
- 目前不该做什么：不要把更多角色、更多 agent 人设、更多花式入口当成核心护城河。
- 当前状态：高置信度。

### 3. 更该先服务 `3-30` 人团队或高自主专业个体，不先服务泛个人市场
- 为什么当前成立：这类用户痛点更真、频率更高、预算更明确，也更愿意为“少漏事、自动推进、交付更稳”付费。泛个人市场容易被免费额度、娱乐型需求和新鲜感吞掉。
- 支持证据：Jack 当前讨论的高价值场景，本质都更像工作流与结果交付问题，不像泛消费聊天需求；这类场景天然更接近团队协作或高自主专业工作者。
- 反证 / 削弱条件：如果后续真实证据表明，泛个人用户才是更快付费、留存更强、传播更快的人群。
- 当前资源含义：巡检优先盯“已有预算但体验很烂”的用户群，不优先盯大众情绪型热点。
- 这周该做什么验证：继续补哪些岗位 / 团队会为会议后跟进、跨端接力、任务推进、知识沉淀直接付钱。
- 目前不该做什么：不要因为大厂大众产品教育了市场，就默认要走大众产品路线。
- 当前状态：候选 thesis，需继续补证据。

### 4. 经营红线：不能滑向 AI 外包
- 为什么当前成立：如果价值主要来自人工兜底、项目制定制和背后人力，规模一上来就会塌。那不是产品，只是披着 agent 外壳的服务外包。
- 支持证据：这条红线已经被长期记忆反复强调；Claw / Agent 赛道很多“能跑通”的体验，本质依赖人工介入、客服兜底或非标准流程。
- 反证 / 削弱条件：只有当大量案例证明，人工重介入依然能稳定复制、毛利可观、体验也能持续优于系统化方案时，这条红线才需要重估。
- 当前资源含义：巡检和产品判断都要主动找“人工兜底比例”“失败后靠谁救”“能力是否沉淀进系统”。
- 这周该做什么验证：继续找热门 agent 产品的失败恢复路径、权限策略、默认配置能力，判断哪些其实在卖隐形人工。
- 目前不该做什么：不要把“有人替你搞定”误判成“系统默认帮你搞定”。
- 当前状态：硬约束，不是可选偏好。
