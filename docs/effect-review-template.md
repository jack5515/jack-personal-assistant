# Effect Review Template

这份模板用于复盘：这套协作到底有没有帮助 Jack 成为一个更强的一人公司。

目标不是复盘“系统有没有跑”，而是复盘“系统有没有产生真实帮助”。

## Review Questions

### 1. 情报层
- 最近 3 天的 AI 巡检日报，有没有带来 Jack 原本不会主动看到的高价值 signal？
- 这些 signal 里，有没有至少 1 条真的影响了产品判断、创业判断或关注方向？
- 如果没有，问题出在：搜索范围、watchlist、阈值、表达方式，还是投递节奏？

### 2. 判断层
- 最近新增的背景知识 / 长期记忆，有没有真的让后续判断更快、更稳？
- 有没有哪条 thesis 被加强、削弱或推翻？
- 有没有新的框架已经开始复用，而不是只写了一次？

### 3. 执行层
- 最近有哪些发送、汇报、同步或自动化动作，真的减少了 Jack 的手动成本？
- 哪些动作虽然完成了，但价值感不高？
- 哪些动作应该停、应该降频、应该换格式？

### 4. 总体结果
- 这 3 天里，系统最像“超强一人公司的外脑/执行系统”的地方是什么？
- 最不像的地方是什么？
- 下一轮只做哪 1-2 个最关键升级，会最直接帮助 Jack 更强？

## Output Format

- `有效帮助`
- `低效动作`
- `最新判断`
- `下一轮只做什么`

## Rule

默认短、直接、只讲有用的。
不要为了复盘而复盘。
