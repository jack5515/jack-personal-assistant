# Effect Review Template

这份模板用于复盘：这套协作到底有没有帮助 Jack 更快逼近值得做的创业命题，而不是只是看了更多信息。

目标不是复盘“系统有没有跑”，而是复盘“系统有没有带来真实决策增量和创业动作”。

## Review Questions

### 1. 收敛层
- 最近 `3` 天的巡检，有没有让某条 thesis 明显变强、变弱或被推翻？
- 最近 `3` 天的巡检，有没有让 Jack 更接近一个明确的楔子、用户或付费点？
- 如果没有，问题出在：搜索范围、watchlist、阈值、结构，还是根本没找到真实用户证据？

### 2. 动作层
- 最近的输出，有没有至少触发 `1` 个具体动作：约人聊、补证据、打 MVP、改口径、停某条路线？
- 哪些输出看起来判断不错，但没有转成动作？
- 无强 signal 的日子，系统有没有自动补 `反证 / 候选机会拆解 / 关键证据缺口`，还是只说“今天没大事”？

### 3. 过滤层
- 最近被过滤掉的内容里，有没有其实应该进来的高价值信号？
- 最近发出来的内容里，有没有本质上只是“高质量新闻整理”？
- 哪些来源继续值得加码，哪些来源该降权或直接停掉？

### 4. 总体结果
- 这 `3` 天里，系统最像“创业驾驶舱”的地方是什么？
- 最不像的地方是什么？
- 下一轮只做哪 `1-2` 个升级，会最直接帮助 Jack 更快收敛？

## Output Format

- `最强增量`
- `被削弱的 thesis`
- `本周动作`
- `应该停止的事`

## Rule

默认短、直接、只讲有用的。
不要为了复盘而复盘。
