# Custom Agents

This directory contains Jack-specific custom agents built around recurring workflows, judgment patterns, and monitoring needs.

## Agent List

- `briefing-scout.md`
- `pm-radar.md`
- `market-watch.md`
- `signal-skeptic.md`
- `automation-operator.md`
- `knowledge-librarian.md`
- `workflow-architect.md`
- `service-designer.md`

Each file defines:
- mission
- when to use
- when not to use
- inputs
- outputs
- working style

Supporting system files:
- `task-routing.md` for task ownership
- `collaboration-map.md` for agent cooperation
- `activation-guide.md` for quick usage
- `decision-policy.md` for conflict resolution
- `task-states.md` for shared status flow
- `runtime-bindings.md` for real workspace bindings
