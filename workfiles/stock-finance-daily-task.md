# Stock Finance Daily Task

目标：每天早上生成一份给 Jack 的股票/财经晨报，并发送到当前飞书私聊。

## 当前约束
- 当前机器未配置 `tushare` / `baostock` / `pandas`
- 当前机器未配置 `TUSHARE_TOKEN`
- 因此现阶段先使用 OpenClaw agent 的检索与公开信息能力，产出一个最小可用版晨报
- 后续若补齐 `tushare + baostock`，再切换到用户偏好的国内数据源版本

## 输出要求
- 用中文输出
- 结构必须包含：`已完成 / 候选 / 低置信度 / 待验证`
- 不得假装有实时行情终端级精度，不给伪精确数字
- 重点服务 Jack：AI 产品经理 + 业余关注股票和财务

## 每日晨报结构
标题：股票财经每日简报 - YYYY-MM-DD

1. 已完成
- 隔夜美股/中概/科技股的主要变化
- 当天值得关注的宏观、政策、财报、行业催化
- 对 A 股/港股/美股可能有影响的 3-5 个重点信号

2. 候选
- 2-4 个值得继续跟踪的板块/主题/个股线索
- 每条给一句“为什么值得看”

3. 低置信度
- 市场传闻、未证实消息、推测性判断
- 必须明确标低置信度原因

4. 待验证
- 今天开盘前/盘后最值得补核实的 2-3 个点

## 分析偏好
- 优先关注：美股科技、AI、半导体、中概、政策、流动性、财报、风险偏好
- 如果当日缺少高质量股票信号，允许多写财经/宏观，不强行凑个股
- 不给投资建议式口吻，优先给“观察框架 + 风险点”

## 风格
- 简洁
- 直接
- 有判断，但不装确定
- 对不确定信息明确标注

## 投递
- 直接发送到 Jack 当前飞书私聊
- 最终回复只发晨报正文，不要过程说明
